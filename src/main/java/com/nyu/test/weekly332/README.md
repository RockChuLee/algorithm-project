# 反思
1. 其实应该尽量少使用java自带的一些api，多关注一些原生的逻辑，比如Integer.parseInt
2. 如果答案和数组的顺序无关，就可以使用排序
3. 双重便利的时间复杂度是O(n2)，如果数组是有序的，可以使用二分查找，将时间复杂度降至O(nlogn)
4. val ^ first = second  - > val = second ^ first
5. 2^31 - 1 > 2^30 > 10^9

2147483647 = 2^31 - 1
1073741823 = 2^30
1000000000 = 10^9
6. 将 二进制码 转换成 十进制

x = x << 1 | (s[r] & 1);
